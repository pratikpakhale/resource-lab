# Resource Lab

Manage your links w/ the help of OpenAI and NotionAPI

## Deployment

Site is live at - [resource-lab.netlify.app](https://resource-lab.netlify.app/)

## Installation and Run

Install with yarn

```bash
  yarn
  yarn dev
```

also, in a parallel terminal run -

```bash
yarn dev:server
```

## API Keys

| Requirements             |
| :----------------------- |
| `Notion Integration Key` |
| `Notion Database ID`     |
| `OpenAI Key`             |
